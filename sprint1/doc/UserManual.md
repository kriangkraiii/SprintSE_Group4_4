# คู่มือการใช้งาน Sprint 1
### “Ride” เว็บแอปพลิเคชันการเดินทางร่วมกันอย่างปลอดภัย


## 1. ระบบบันทึกประวัติการใช้งานตามข้อกำหนดทางกฎหมาย (System Log) (PBL1)
ระบบมีการจัดเก็บข้อมูลจราจรทางคอมพิวเตอร์ที่สอดคล้องกับข้อกำหนดทางกฎหมายตามพระราชบัญญัติว่าด้วยการกระทำความผิดเกี่ยวกับคอมพิวเตอร์ (ฉบับที่ 2) พ.ศ. 2560 มาตรา 26 โดยระบบจะจัดเก็บข้อมูลจราจรทางคอมพิวเตอร์ไว้ไม่น้อยกว่า 90 วัน เพื่อใช้เป็นหลักฐานสำคัญในการตรวจสอบและสืบค้นเหตุการณ์ย้อนหลัง

### ขั้นตอนการเข้าใช้งานสำหรับผู้ดูแลระบบ
1. **การเข้าสู่ระบบ:** กรอกชื่อผู้ใช้หรืออีเมล และรหัสผ่านที่หน้า Login จากนั้นกดปุ่ม “เข้าสู่ระบบ”
2. **หน้า Dashboard:** เมื่อเข้าสู่ระบบเรียบร้อยแล้ว นำเมาส์ไปวางที่ Profile ด้านบนขวาแล้วกดที่เมนู “Dashboard”
3. **ดู Log:** เลือก “System Logs” ที่แถบเมนูด้านซ้าย โดยจะแสดงข้อมูลต่าง ๆ ได้แก่ ID, Timestamp, User ID, IP Address, Action (get, post, put, delete), Resource และ User Agent
4. **ค้นหาข้อมูล:** User ID, IP Address, Action, ตั้งแต่วันที่, ถึงวันที่ แล้วกดปุ่ม “ค้นหา” เพื่อทำการค้นหาข้อมูล

## 2. การจัดการบัญชีดำ (Blacklist Management) (PBL3)
ผู้ดูแลระบบสามารถระงับสิทธิ์ผู้ใช้งานที่ทำผิดกฎหรือมีพฤติกรรมที่ไม่เหมาะสม เพื่อป้องกันไม่ให้ผู้ใช้งานคนดังกล่าวกลับมาใช้งานระบบได้อีกครั้ง ซึ่งสอดคล้องกับข้อกำหนดทางกฎหมายตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) มาตรา 22 โดยระบบใช้การ Data Minimization ด้วยการจัดเก็บหมายเลขบัตรประชาชนในรูปแบบ National ID Hash (SHA-256) เพื่อคุ้มครองข้อมูลส่วนบุคคลของผู้ใช้งานที่ถูกระงับสิทธิ์
### ขั้นตอนการเข้าใช้งานสำหรับผู้ดูแลระบบ
1.	**การเข้าสู่ระบบ:** กรอกชื่อผู้ใช้หรืออีเมล และรหัสผ่านที่หน้า Login จากนั้นกดปุ่ม “เข้าสู่ระบบ”
2.	**หน้า Dashboard:** เมื่อเข้าสู่ระบบเรียบร้อยแล้ว นำเมาส์ไปวางที่ Profile ด้านบนขวาแล้วกดที่เมนู “Dashboard”
3.	**ดู Blacklist:** เลือก “Blacklist” ที่แถบเมนูด้านซ้าย เพื่อดูข้อมูลรายชื่อผู้ใช้งานที่ถูกระงับสิทธิ์
4.	กดปุ่ม “+ เพิ่ม Blacklist” ปุ่มสีแดงด้านขวาบน
5.	กรอกหมายเลขบัตรประชาชน 13 หลัก และระบุเหตุผลในการระงับสิทธิ์ 
6.	กดปุ่ม “+ เพิ่ม Blacklist” เพื่อยืนยันข้อมูลการบันทึก

## 3. ระบบยืนยันตัวตนอัตโนมัติ (Automated User Verification) (PBL4)
ระบบถูกพัฒนาขึ้นเพื่อช่วยลดภาระของผู้ดูแลระบบและเพิ่มประสิทธิภาพในการทำงาน โดยการนำเทคโนโลยี IAPP OCR (Optical Character Recognition) เข้ามาใช้ในการอ่านและสกัดข้อมูลจากรูปภาพ เช่น บัตรประชาชน เพื่อนำไปตรวจสอบความถูกต้องกับข้อมูลในระบบโดยอัตโนมัติ ช่วยลดข้อผิดพลาดจากการที่ผู้ใช้งานกรอกข้อมูลด้วยตนเอง
### ขั้นตอนการยืนยันตัวตนอัตโนมัติในส่วนของบัตรประชาชน
1.	**การสมัครสมาชิก (ขั้นตอนที่ 1 บัญชีผู้ใช้):** กรอกชื่อผู้ใช้ อีเมล รหัสผ่าน และยืนยันรหัสผ่าน 
2.	**การสมัครสมาชิก (ขั้นตอนที่ 2 ข้อมูลส่วนตัว):** กรอกชื่อจริง นามสกุล เบอร์โทรศัพท์ และเพศ
3.	**การสมัครสมาชิก (ขั้นตอนที่ 3 ยืนยันตัวตน):** ภาพถ่ายบัตรประชาชนด้านหน้า ภาพถ่ายบัตรประชาชนด้านหน้า และภาพถ่ายตนเองคู่กับบัตรประชาชน 
4.	**ตรวจสอบข้อมูล:** กดปุ่มตรวจสอบข้อมูลแต่ละส่วน หากข้อมูลตรวจสอบผ่านจะแสดงข้อมูลที่อ่านได้จากบัตรประชาชน แต่ถ้าหากข้อมูลตรวจสอบไม่ผ่านจะแสดงข้อความแจ้งเตือน
5.	**ยอมรับเงื่อนไข:** อ่านข้อตกลงการใช้บริการและนโยบายความเป็นส่วนตัว จากนั้นกด Checkbox ยอมรับเงื่อนไขดังกล่าว และกด “สมัครสมาชิก” 
### ขั้นตอนการยืนยันตัวตนผู้ขับขี่ (Driver Verification)
1.	**การเข้าสู่ระบบ:** กรอกชื่อผู้ใช้หรืออีเมล และรหัสผ่านที่หน้า Login จากนั้นกดปุ่ม “เข้าสู่ระบบ”
2.	**หน้าบัญชีของฉัน:** เมื่อเข้าสู่ระบบเรียบร้อยแล้ว นำเมาส์ไปวางที่ Profile ด้านบนขวาแล้วกดที่เมนู “บัญชีของฉัน”
3.	กด “การยืนยันตัวตนสำหรับผู้ขับขี่” ที่แถบเมนูทางด้านซ้าย
4.	กดปุ่ม “ยืนยันตัวตนด้วย OCR (อัตโนมัติ)” 
5.	อัปโหลดรูปใบขับขี่ด้านหน้าและรูปถ่ายตนเองคู่กับใบขับขี่ 
6.	กดปุ่ม “ตรวจสอบใบขับขี่ด้วย ORC” หากข้อมูลตรวจสอบผ่านจะแสดงข้อมูลที่อ่านได้จากบัตรประชาชน แต่ถ้าหากข้อมูลตรวจสอบไม่ผ่านจะแสดงข้อความแจ้งเตือน
7.	เมื่อตรวจสอบข้อมูลผ่าน ORC เรียบร้อยแล้ว กดปุ่ม “ยืนยันตัวตนและสร้างบัญชีคนขับ”

## 4. ระบบลบบัญชีและข้อมูลส่วนตัว (Delete Account) (PBL16)
ถูกพัฒนาขึ้นมาเพื่อให้ผู้ใช้งานสามารถลบบัญชีและข้อมูลส่วนตัวของตนเองได้เมื่อต้องการยกเลิก การเป็นสมาชิก ซึ่งสอดคล้องกับพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) มาตรา 33 ที่อนุญาตให้เจ้าของข้อมูลสามารถขอให้ลบข้อมูลส่วนบุคคลของตนเองได้เมื่อหมดความจำเป็นในการใช้งาน และยังสอดคล้องกับพระราชบัญญัติว่าด้วยการกระทำความผิดเกี่ยวกับคอมพิวเตอร์ พ.ศ. 2560 มาตรา 26 ซึ่งมีเงื่อนไขว่าต้องจัดเก็บข้อมูลจราจรทางคอมพิวเตอร์ต่อไปอีกไม่น้อยกว่า 90 วัน เพื่อนำไปใช้เป็นหลักฐานสำคัญในการตรวจสอบและสืบค้นเหตุการณ์ย้อนหลัง
### ขั้นตอนการลบบัญชี
1.	**การเข้าสู่ระบบ:** กรอกชื่อผู้ใช้หรืออีเมล และรหัสผ่านที่หน้า Login จากนั้นกดปุ่ม “เข้าสู่ระบบ”
2.	**หน้าบัญชีของฉัน:** เมื่อเข้าสู่ระบบเรียบร้อยแล้ว นำเมาส์ไปวางที่ Profile ด้านบนขวาแล้วกดที่เมนู “บัญชีของฉัน”
3.	กด “โปรไฟล์ของฉัน” และเลื่อนลงมาด้านล่างสุด 
4.	กดปุ่ม “ลบบัญชีของฉัน”
